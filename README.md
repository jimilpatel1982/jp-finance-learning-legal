# JP Finance Learning Lab – Legal

This repository contains **public legal documents** for mobile applications developed under the **JP Finance Learning Lab** name.

It exists solely to host **privacy policy and related legal pages** on a publicly accessible URL, as required by app stores (e.g., Google Play).

---

## 📄 What is in this repository?

- `docs/privacy.html`  
  A static HTML version of the Privacy Policy for the **MoneyMap App**.

This page is designed to be:
- Human‑readable
- Publicly accessible
- Google Play Store compliant

---

## 🔒 What is NOT in this repository?

- ❌ Application source code
- ❌ Business logic
- ❌ APIs or backend services
- ❌ User data
- ❌ Tracking or analytics

All application code lives in a **separate private repository**.

---

## ✅ Why a separate repository?

Maintaining legal pages separately allows:

- Public access to privacy policies (store compliance)
- Private ownership of application source code
- Clear separation between product logic and legal documentation

This repository should not be treated as an open‑source project.
